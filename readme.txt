ZenYoTech - Site vitrine IA Générative

Présentation :
Ce site d'actualité est dédié à l'intelligence artificielle générative et aux modèles de langage récents.

Structure du site :
- index.html : page d’accueil avec affichage des 10 articles les plus récents
- archives.html : liste complète des articles archivés
- about.html : présentation du site et de sa mission
- contact.html : formulaire simple de contact
- style.css : design zen high-tech, clair/sombre, responsive
- script.js : gestion de la date/heure, thème, import des articles JSON
- data/articles.json : fichier à mettre à jour hebdomadairement avec les articles
- assets/logo.svg, favicon.ico, images défaut : ressources graphiques

Hébergement recommandé :
- Netlify (https://www.netlify.com)
- GitHub Pages (https://pages.github.com)
- Vercel (https://vercel.com)

Instructions déploiement :
1. Placez tous les fichiers dans un dossier unique
2. Publiez ce dossier sur votre plateforme d'hébergement statique préférée
3. Mettez à jour hebdomadairement le fichier /data/articles.json pour rafraîchir les articles
4. Le site affichera automatiquement les articles les plus récents en page d'accueil

Pour toute modification :
- Editez les fichiers HTML, CSS ou JS selon vos besoins
- Ajoutez ou modifiez images dans le dossier assets/
